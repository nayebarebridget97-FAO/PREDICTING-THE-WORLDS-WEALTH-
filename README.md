# FAO-ECONOMIC--ML--PROJECT
Analysis and Machine Learning on FAO Macroeconomic data 
Project Description
This project analyzes global economic and agricultural data from the FAO dataset.

It focuses on understanding how agriculture contributes to the economy and how different economic indicators change over time across different countries.

🎯 Objectives
- Understand and explore the dataset
- Visualize key economic and agricultural trends
- Identify relationships between variables
- Build machine learning models to make predictions

#Dataset Information
The dataset includes:
- Countries (Area)
- Years (1970–2023)
- Economic indicators (e.g., GDP)
- Agricultural indicators
- Measured values

## Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib & Seaborn
- Scikit-learn
- TensorFlow / Keras
