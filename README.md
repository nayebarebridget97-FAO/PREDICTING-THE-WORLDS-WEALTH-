# FAO-ECONOMIC--ML--PROJECT
## 📖 Project Overview
I built a computer "brain" (Artificial Intelligence) that looks at how much money a country makes from Agriculture and uses that to guess the country's total wealth (GDP).
 
Project Description
This project analyzes eceomies using agricultural data from the FAO dataset.
It focuses on understanding how agriculture contributes to the economy and how this changes over time across different countries.

🎯 Objectives
- Understand and explore the dataset
- Visualize key economic and agricultural trends
- Identify relationships between different variables
- Build machine learning models to make GDP predictions

#Dataset Information
The dataset includes:
- Countries (Area)
- Years (1970–2023)
- Economic indicators (e.g., GDP, Value Added Agriculture, etc )
- Measured values

## Tools Used
- Python
- Google Colab

- Matplotlib & Seaborn
- Scikit-learn

##Models used
-Random Regressor: A classic method that acts like a team of experts making a decision together.
-Neural Network: A modern method inspired by how the human brain works.
